# databases_course_work
Курсовая работа по дисциплине "Базы данных" на тему "Разработка базы данных для автоматизации учета в клубе любителей собаководства" студента гр. Б24-191-2 Карандашовой М.В.
Файл create_tables.sql содержит sql запросы для создания структуры базы данных и первичного заполнения ее данными.
Файл main.py содержит код на языке Python, реализующий пользовательское приложение для работы с разработанной базой данных.
